full web app from scratch: an Event / Booking System.

Core Features  
- Event listings with details (name, description, date/time, venue, images).  
- Users can view events and book tickets / seats.  
- Authentication: user accounts; booking history.  
- Organizer role: create events, manage bookings, capacity limits.  
- Payment flow (mock or real).  
- Notifications / confirmations.  

UI / UX Requirements  
- List view of upcoming events; event detail page.  
- Calendar or timeline view.  
- Booking UI: seat selection (if relevant) or ticket quantities.  
- Responsive layout; theme toggle.  

Technical & Integrations  
- Backend for users, events, bookings.  
- File/image storage.  
- Payment gateway.  
- Email or notification simulation.  

Bonus / Advanced  
- Seat map / seating layout.  
- Organizer dashboard with analytics (attendance, revenue).  
- Waiting list when event is full.  
- Share events / social media integration.  

Final app: event discovery, booking, management, UI + backend.
